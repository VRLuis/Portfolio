# Portfolio

This repository contains a personal portfolio website built with Astro and TypeScript.

The site presents:

- A localized experience with English and Spanish support.
- A home section with a personal introduction and navigation.
- A projects section showcasing portfolio work.
- An experience section listing professional roles and achievements.
- A formation and certifications section with education and certificates.
- A learning section that highlights ongoing skills and study areas.
- Contact information and social links.

The project structure includes reusable Astro components for atoms, molecules, organisms, and sections, along with data files for bilingual content.
